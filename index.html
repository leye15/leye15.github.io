Data Mining - Prédiction du Risque de Récidive

Projet complet de Data Mining visant à prédire le risque de récidive chez les délinquants. Nettoyage, exploration, imputation et modélisation sur un jeu de données de 1.4 million d'observations provenant du système judiciaire.

Python Pandas NumPy Scikit-learn Data Mining Machine Learning

Contexte et Objectifs

Ce projet aborde une problématique cruciale du système judiciaire : la prédiction du risque de récidive. À partir d'un dataset massif contenant 54 variables décrivant des cas judiciaires (démographie, antécédents, types d'infractions, décisions de justice), l'objectif est de développer un modèle prédictif fiable pour identifier les facteurs clés influençant la récidive.

Méthodologie et Processus

1

Analyse Exploratoire des Données

Examen approfondi du dataset initial : identification des types de variables, détection des valeurs aberrantes, analyse des distributions et des corrélations. Calcul du pourcentage de valeurs manquantes par variable.

2

Nettoyage et Prétraitement

Correction de la cohérence temporelle (âge jugement ≥ âge infraction), suppression des valeurs aberrantes, conversion des types de données. Création de 4 variables dérivées pertinentes pour la modélisation.

3

Imputation Intelligente des Valeurs Manquantes

Stratégie d'imputation basée sur la logique métier : jail = 0 quand probation = 1, imputation par la médiane conditionnelle (case_type × violent_crime), imputation du mode pour les variables catégorielles.

4

Modélisation Prédictive

Construction de modèles de classification (régression logistique, random forest, XGBoost) pour prédire recid_180d. Optimisation des hyperparamètres par validation croisée.

Détails Techniques du Projet Data Mining

Préparation des Données
  • Dataset : 1,476,967 observations, 54 variables initiales
  • Nettoyage : Suppression des valeurs aberrantes (âge < 7 ou > 100 ans)
  • Imputation : Logique métier pour les valeurs manquantes
  • Variables dérivées : 4 nouvelles variables créées
Stratégie d'Imputation
  • Probation = 1 : jail = 0 (pas de prison)
  • Release = 1 : jail = 0 (libération)
  • Autres cas : Médiane par case_type × violent_crime
  • Variables catégorielles : Imputation par le mode
Analyse Statistique
  • Tests de Mann-Whitney : Variables numériques vs probation
  • Tests du Chi² : Variables catégorielles vs probation
  • Significativité : Toutes les p-values < 0.0001
  • Corrélations : Analyse des relations entre variables

Résultats Clés
  • Meilleur modèle : XGBoost avec AUC = 0.83
  • Variables importantes : antécédents, gravité des infractions, âge
  • Impact potentiel : Aide à la décision judiciaire

Modélisation Statistique Avancée avec Régularisation Lasso

Projet complet de modélisation statistique appliquant les techniques de régularisation Lasso sur différents types de modèles : régression logistique, régression linéaire et modèles de survie. Ce projet démontre ma maîtrise des méthodes avancées de sélection de variables et de prévention du surajustement.

R glmnet survival ggplot2 Validation croisée Sélection de variables

Contexte et Objectifs

La régularisation Lasso (Least Absolute Shrinkage and Selection Operator) est une technique puissante pour la sélection de variables et la prévention du surajustement dans les modèles statistiques. Ce projet explore l'application du Lasso sur trois types de modèles différents, avec une évaluation comparative des performances.

Structure du Projet

Partie 1 : Régression Linéaire avec Lasso
  • Application du Lasso sur des données continues
  • Optimisation du paramètre lambda par validation croisée
  • Sélection automatique des variables prédictives
  • Comparaison avec la régression linéaire classique
Partie 2 : Régression Logistique avec Lasso
  • Classification binaire avec régularisation L1
  • Évaluation des performances par matrice de confusion
  • Courbes ROC et analyse AUC
  • Application sur des données médicales
Partie 3 : Modèles de Survie avec Lasso
  • Extension du Lasso au modèle de Cox
  • Analyse des courbes de Kaplan-Meier
  • Validation des hypothèses des risques proportionnels
  • Application sur des données de survie

Détails Techniques de la Régularisation Lasso

Principe du Lasso
  • Régularisation L1 : Pénalisation de la somme des valeurs absolues des coefficients
  • Sélection de variables : Mise à zéro des coefficients non pertinents
  • Prévention du surajustement : Réduction de la variance au prix d'un biais modéré
  • Paramètre lambda : Contrôle l'intensité de la régularisation
Optimisation du Paramètre Lambda
  • Validation croisée : 10-fold cross-validation
  • Critère d'optimisation : Minimisation de l'erreur de prédiction
  • Lambda.min : Valeur minimisant l'erreur de validation
  • Lambda.1se : Valeur la plus parcimonieuse dans l'intervalle de confiance
Avantages du Lasso
  • Sélection automatique : Élimination des variables non informatives
  • Interprétabilité : Modèles plus simples et plus compréhensibles
  • Performance : Meilleure généralisation sur de nouvelles données
  • Multicollinéarité : Gestion efficace des variables corrélées
